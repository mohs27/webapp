# Installation

## Development

:::info
You need [Node.js](https://nodejs.org/en/) to build it
:::

HoloPlay is a JavaScript application that runs in the browser.

First, install project dependencies :

```bash
npm i
```

Then, run the development server :

```bash
npm start
```

That it ! You can now access at [localhost:3000](http://localhost:3000).
